# v2.0.0

**Release Date:** February 2026

**Platforms:** iOS (TestFlight) | Android (Play Store)

---

## Overview

TrickBook v2.0.0 is a complete rebuild of the mobile app using React Native 0.81, Expo SDK 54, TypeScript, NativeWind, and Zustand. This release delivers a modernized UI, new social features, and a fully redesigned user experience.

## What's New

### Personalized Home Screen
A new dashboard with current goals, progress stats, quick actions, and homie activity — all in one place.

![Home Screen](/img/releases/v2.0/theHome.png)

### The Feed — Share Clips, Hype Up Homies
Full-screen vertical video feed for sharing skate clips. React with likes, comments, and emoji reactions. Tag tricks and hashtags on every post.

![The Feed](/img/releases/v2.0/theFeed.png)

### Trickipedia — Lookup New Tricks
Browse the full trick encyclopedia with video previews. Filter by difficulty and add tricks directly to your personal TrickList.

![Trickipedia](/img/releases/v2.0/theTrickipedia.png)

### Find & Save Spots
Discover skate spots on an interactive map. Filter by parks and street spots, view ratings, and save your favorites.

![Spots](/img/releases/v2.0/theSpots.png)

### Connect With Homies
Find and connect with other skaters. Search for homies, send requests, and message directly within the app.

![Homies](/img/releases/v2.0/theHomies.png)

## Technical Highlights

- **React Native 0.81** with **Expo SDK 54** and dev client
- **TypeScript** throughout the entire codebase
- **NativeWind** (Tailwind CSS) for styling
- **Zustand** for state management (replaced Redux)
- **Session persistence** fixed for iOS with secure token storage
- **Google Maps API key** secured via environment variables and EAS secrets
- **EAS Build** profiles for TestFlight, Play Store, development, and preview

## Known Issues

- Android Play Store build is in internal track (draft)
- Real-time messaging via Socket.io is in progress
